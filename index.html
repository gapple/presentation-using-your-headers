X-XSS-Protection

Chrome, Safari, IE (no FF)
X-XSS-Protection
  • 0
  • 1
  • 1; mode=block
  • 1; report=<reporting-uri>
`1`: remove unsafe parts (not recommended; can be tricked to remove legitimate code) `1; mode=block`: prevent page rendering report is Chrome only
